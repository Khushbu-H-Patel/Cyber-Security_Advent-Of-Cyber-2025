# Network Discovery - Scan-ta Clause

This room explains the network service discovery with nmap to identify and exploit the open ports and services in the server and find out way to gain access and kick out the bad actors. Also talks about the netcat (nc) command to interact with the network services and dig command which is used to perform advanced DNS queries.
